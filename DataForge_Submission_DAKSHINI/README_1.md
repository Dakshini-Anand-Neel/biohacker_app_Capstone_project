# DataForge Submission Folder Structure

This document summarizes the folder structure and primary contents for the `DataForge_Submission_DAKSHINI` workspace.

## Root Folder

- `README.md` - Project overview and app deployment instructions
- `README_1.md` - Folder structure summary (this file)
- `Capstone_WebApp_Deployment/` - Streamlit app deployment folder
- `Practical_01_Excel_Cleaning/` - Practical 1 Excel cleaning report
- `Practical_02_Excel_Stats/` - Practical 2 Excel statistics report and workbook
- `Practical_03_Excel_Pivot/` - Practical 3 pivot/dashboard preview and report
- `Practical_04_PBI_PowerQuery/` - Practical 4 Power BI Power Query M code and report
- `Practical_05_PBI_DAX/` - Practical 5 DAX formulas, schema image, and report
- `Practical_06_PBI_Dashboard/` - Practical 6 dashboard export and report
- `Practical_07_Python_Pandas/` - Practical 7 Python Pandas report
- `Practical_08_Python_Wrangling/` - Practical 8 Python wrangling report
- `Practical_09_Python_Stats/` - Practical 9 Python statistics report
- `Practical_10_Python_Visuals/` - Practical 10 Python visuals report

## Capstone_WebApp_Deployment

- `.gitignore`
- `Biohacker_Habit_&_Sleep.py` - Main dashboard application logic
- `README.md` - Deployment and app overview
- `requirements.txt` - Python dependency list
- `streamlit_app.py` - Streamlit entry point
- `venv/` - Local Python virtual environment
- `__pycache__/` - Python bytecode cache
